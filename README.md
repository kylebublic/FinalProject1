This is my final app team project, which is called Wiki Guessr
Wiki Guessr was inspired by boredom and a love for Wikipedia,
It will allow users to test their common sense by guessing what Wikipedia page each image originated from
I have developed this project using WikiData API provided by Wikipedia

Along the way, I added a simple themes screen as well as a 'streaks' system to keep count of your scores. In the future, I would like to improve and polish the UI
Polishing could take the form of a leaderboard, further themes outside of basic colors, prizes or messages for reaching certain score milestones, or more

One obstacle that I encountered was when implementing a filter to only display Wikipedia pages that included thumbnail images.
Thankfully, Wikipedia's robust API allowed for this and made implementing this filter relatively easy by defining pages with a ThumbnailURL in the service page.
